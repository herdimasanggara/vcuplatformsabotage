How to create a square in CSS?


A square is simply a box with equal width and height. Here's the code to create a square:


.square { width: 100px; height: 100px; background-color: lightgray; }

How does the animation work?


This square uses a CSS animation to move from left to right. The @keyframes rule defines the movement, and the animation is applied with the animation property:


@keyframes move { 0% { transform: translateX(0); } 100% { transform: translateX(200px); } } .square { animation: move 5s infinite alternate; }

How to create a circle in CSS?


A circle is created by setting the width and height of a box equally and applying a border-radius of 50%. Try this:


.circle { width: 100px; height: 100px; background-color: yellow; border-radius: 50%; }

How does the animation work?


The circle uses a color-changing animation. The @keyframes rule specifies the change from yellow to lightgray, and the animation is applied with the animation property:


@keyframes colorChange { 0% { background-color: yellow; } 100% { background-color: lightgray; } } .circle { animation: colorChange 3s infinite; }

How to create a triangle in CSS?


A triangle can be made using borders. The following code creates an upward-pointing triangle:


.triangle { width: 0; height: 0; border-left: 50px solid transparent; border-right: 50px solid transparent; border-bottom: 100px solid black; }

How does the animation work?


The triangle uses a shaking animation. The @keyframes rule defines slight movements to the left and right, creating a shake effect:


@keyframes shake { 0%, 100% { transform: translateX(0); } 50% { transform: translateX(-10px); } } .triangle { animation: shake 1s infinite; }

How to combine multiple shapes in CSS?


By using CSS positioning, you can layer and combine different shapes to create more complex visuals. Here's an example where we combine a circle and a square to form a new shape:


.combined-container { position: relative; width: 150px; height: 150px; } .combined-square { width: 100px; height: 100px; background-color: lightgray; position: absolute; top: 25px; left: 25px; } .combined-circle { width: 100px; height: 100px; background-color: yellow; border-radius: 50%; position: absolute; top: 0; left: 0; }
