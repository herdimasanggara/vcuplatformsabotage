Go ahead and open a new file. You can do this by clicking on File > New File or using Ctrl+N (Cmd+N on macOS). Save it right away as index.html to keep things organized. Just hit File > Save As....
Now, open up your new index.html file. Here's a quick boilerplate to get you startedâ€”copy this in:
<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>My First HTML Page</title> </head> <body> <h1>Hello, World!</h1> <p>This is my first webpage.</p> </body> </html>
Remember to save your progress as you go with Ctrl+S (Cmd+S on macOS).
Let's make it your own! Change the text inside the <h1> and <p> tags to something unique to you. For example:
<body> <h1>Welcome to My Website</h1> <p>I'm learning how to build webpages!</p> </body>
Don't forget to save again!
Open the folder where you saved index.html. Double-click it to open in your web browser and check out your work.
In VSCode, you can quickly set up a basic HTML structure by typing '!' and pressing Enter.
Remember to save your work often, especially before previewing your webpage. If you don't save, any changes you've made won't show up!
