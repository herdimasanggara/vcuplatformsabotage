09/04, HTML (2/2)

For this exercise, we will introduce new HTML tags: <table>, <col>, <img>, and <audio>. You will also use inline styles to modify background and text colors. Start by creating a new HTML file named htmlexp3.html with this basic structure:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>HTML Experiment (3/3)</title> </head> <body> <!-- Your new experiment will go here --> </body> </html>

Create a simple table with <table>, <tr>, and <td> tags to structure information. You can use <col> inside a <colgroup> to style columns. For example:

<table style="width:50%; background-color:white; color:black;"> <colgroup> <col span="2" style="background-color:yellow;"> </colgroup> <tr> <td>Row 1, Cell 1</td> <td>Row 1, Cell 2</td> <td>Row 1, Cell 3</td> </tr> <tr> <td>Row 2, Cell 1</td> <td>Row 2, Cell 2</td> <td>Row 2, Cell 3</td> </tr> </table>
Row 1, Cell 1 Row 1, Cell 2 Row 1, Cell 3
Row 2, Cell 1 Row 2, Cell 2 Row 2, Cell 3

Add an image using the <img> tag and experiment with inline styles to change its size and border. For example:

<img src="image.jpg" style="width:200px; height:auto; border:2px solid black;">

Embed audio using the <audio> tag and provide controls for play and pause:

<audio controls> <source src="audio.mp3" type="audio/mpeg"> </audio>

To create a scrolling text effect, use the <marquee> tag. You can customize its behavior, direction, and speed:

<marquee behavior="scroll" direction="left" scrollamount="5">This is scrolling text! </marquee>
This is scrolling text!

You can also make the text bounce back and forth using the behavior="alternate" attribute:

<marquee behavior="alternate" direction="left" scrollamount="5">Bouncing text!</marquee>
Bouncing text!

Try combining these elements in creative ways. How can tables and images interact? What textures or visual hierarchies can you create using different tags and inline styles?

Experiment with the new tags and observe how they change the visual layout and interaction of your webpage.

<table>: Defines a table to display data in rows and columns.

<col>: Specifies column properties for each column within a <colgroup>.

<img>: Embeds an image in the webpage.

<audio>: Embeds audio content with optional controls.

⚠️ We will only use HTML inline styles during this time. In the next session, we will cover CSS, which is the recommended way to handle styling in modern web development.

⚠️ The <marquee> tag is deprecated in the majority of browsers. This means it may not display properly or at all, depending on the browser you are using, as many modern browsers no longer support it.
