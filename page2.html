08/28, HTML (1/2)

For the first exercise, you'll only use <pre>. Start by creating a new HTML file named htmlexp1.html with this basic structure:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Form Art (1/3)</title> </head> <body> <!-- Your ASCII art will go here --> </body> </html>

Next, use the <pre> tag to create preformatted text that maintains spacing and line breaks exactly as you type them. This is ideal for ASCII art. For example:

<pre> , , /(____)\ ( o o ) | (__) | \ '--' / | | </pre>

Try creating your ASCII art without the <pre> tag and using <br> instead to see how it affects the formatting. Now, expand your ASCII art by experimenting with additional HTML tags to create textures and patterns. Create a new HTML file named htmlexp2.html and use the following basic structure:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Form Art (2/3)</title> </head> <body> <!-- Your textured art will go here --> </body> </html>

Use heading tags <h1> to <h6> to vary text sizes and create visual hierarchy. For example:

<h1>Big Text</h1> <h2>Slightly Smaller</h2> <h3>Even Smaller</h3> <h4>And Smaller</h4> <h5>Almost Tiny</h5> <h6>Smallest</h6>

Combine different text styles like <u> for underline, <i> for italics, <b> for bold, and <s> for strikethrough to create more textures:

<p><u>Underlined Text</u></p> <p><i>Italicized Text</i></p> <p><b>Bold Text</b></p> <p><s>Strikethrough Text</s></p>

Mix and match these tags to create unique patterns and observe how they relate to the imagined object or concept.

Remember to experiment with different combinations to see what textures and patterns you can create.

<br>: Inserts a line break.

&nbsp;: Adds a non-breaking space, which is a space that prevents automatic line breaks at its position.

<pre>: Preserves whitespace and line breaks in the enclosed text, making it ideal for preformatted text like ASCII art.

<h1> to <h6>: Heading tags that define the size and importance of text. <h1> is the largest, while <h6> is the smallest.

<u>: Underlines the enclosed text.

<i>: Italicizes the enclosed text.

<b>: Makes the enclosed text bold.

<s>: Strikes through the enclosed text, indicating that it is incorrect or no longer relevant.
